üéØ Challenge Overview

Integrate genomic and transcriptomic data from TCGA to identify cancer subtypes and predict treatment response. Use real public data to discover molecular patterns that could guide precision oncology.

The Mission

Goal: Identify molecular subtypes of cancer using multi-omics integration

Data: Mutation + Expression data from TCGA

Patients: ~50-100 cancer samples

Deliverable: Subtype classification with clinical relevance

üì• Dataset - TCGA Public Data

Download from TCGA via GDC Portal

library(TCGAbiolinks)

# Download mutation data
query_mut <- GDCquery(
    project = "TCGA-BRCA",
    data.category = "Simple Nucleotide Variation",
    data.type = "Masked Somatic Mutation",
    access = "open"
)
GDCdownload(query_mut)
mutations <- GDCprepare(query_mut)

# Download expression data
query_exp <- GDCquery(
    project = "TCGA-BRCA",
    data.category = "Transcriptome Profiling",
    data.type = "Gene Expression Quantification"
)
GDCdownload(query_exp)
expression <- GDCprepare(query_exp)

1. Data Integration (25 points)

A. Genomic Features

library(maftools)

# Load and analyze mutations
maf <- read.maf("mutations.maf")

# Extract key features:
# - Mutation burden per sample
# - Driver gene mutations (TP53, PIK3CA, etc.)
# - Mutational signatures
# - Pathway-level alterations

# Create mutation matrix
mut_matrix <- mutCountMatrix(maf)

# Oncogenic pathway analysis
oncoplot(maf, genes = top_genes)

B. Transcriptomic Features

library(DESeq2)

# Normalize expression data
dds <- DESeqDataSetFromMatrix(
    countData = expression,
    colData = clinical,
    design = ~ 1
)
vsd <- vst(dds)

# Extract variable genes
var_genes <- apply(assay(vsd), 1, var)
top_var <- names(sort(var_genes, decreasing=TRUE)[1:1000])

C. Integration Strategy

Choose an integration method:

  • Simple concatenation: Combine features into single matrix
  • PCA-based: Reduce each omics, then integrate
  • NMF: Non-negative matrix factorization
  • iClusterPlus: Integrative clustering (R package)

2. Subtype Discovery (30 points)

Clustering Analysis

# Simple integration example
library(ConsensusClusterPlus)

# Combine features
combined_matrix <- rbind(
    scale(mut_matrix),
    scale(assay(vsd)[top_var,])
)

# Consensus clustering
cc <- ConsensusClusterPlus(
    combined_matrix,
    maxK = 6,
    reps = 1000,
    pItem = 0.8,
    pFeature = 1,
    clusterAlg = "hc",
    distance = "euclidean"
)

# Look for clean clusters (high consensus)

Subtype Characterization

For each subtype, identify:

  • Recurrent mutations (which genes are mutated?)
  • Expression signatures (which genes are up/down?)
  • Pathway enrichment (which pathways define subtype?)
  • Clinical associations (survival, grade, stage?)
