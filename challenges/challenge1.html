ğŸ¯ Challenge Overview

You've received sequencing data from a collaborator's failed experiment. The lab claims "the sequencing just didn't work," but you suspect there's more to the story. Your job is to investigate what went wrong and salvage what you can.

The Scenario

A research lab sent you 3 FASTQ files from their RNA-seq experiment studying heat shock response in E. coli. The downstream analysis pipeline failed completely, and they need your expertise to:

  • Identify what went wrong with the sequencing
  • Determine if any data can be salvaged
  • Provide recommendations for future experiments

The lab is counting on you to figure out why their $5,000 sequencing run failed and whether they need to repeat the entire experiment.

ğŸ“‹ Required Deliverables

1. Comprehensive QC Report (30 points)

Create a detailed quality control analysis including:

  • Per-base quality scores
  • Adapter contamination levels
  • Sequence duplication levels
  • GC content distribution
  • Overrepresented sequences
  • Multi-sample comparison

Format: HTML report using MultiQC or equivalent

2. Problem Diagnosis Report (25 points)

Write a clear report (1-2 pages) that answers:

  • What went wrong?
  • Root cause analysis with evidence
  • Which samples are most affected?
  • Severity of each issue identified

Format: PDF or Markdown with embedded figures

3. Cleaned Dataset (25 points)

  • Trimmed/filtered FASTQ files for all samples
  • Before/after QC comparison
  • Cleaning metrics (reads retained, quality improvement)
  • Clear documentation of filtering criteria

4. Recommendations (10 points)

  • Should they proceed with this data?
  • Should they resequence?
  • Future prevention strategies
  • Library preparation improvements

5. Code & Documentation (10 points)

  • All analysis scripts (well-commented)
  • README with step-by-step workflow
  • Tool versions documented
  • Reproducible instructions

ğŸ› ï¸ Recommended Tools

Quality Control

  • FastQC - Per-sample quality metrics
  • MultiQC - Aggregate reports

Read Trimming/Cleaning

  • Trimmomatic - Java-based trimmer
  • Cutadapt - Python-based trimmer
  • fastp - All-in-one tool (recommended for beginners)

Contamination Detection (Optional)

  • BLAST - Identify unknown sequences
  • Kraken2 - Taxonomic classification
  • FastQ Screen - Multi-genome contamination screening
