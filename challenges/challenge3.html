üéØ Challenge Overview

Analyze RNA-seq data from a cancer drug treatment experiment. Find differentially expressed genes, identify affected pathways, and tell the biological story of what's happening at the molecular level.

The Experiment

Cell Line: Human cancer cells (breast or lung)

Treatment: Novel therapeutic compound vs. vehicle control

Time Point: 24 hours post-treatment

Replicates: 3 control, 3 treated

Your Job: What is this drug doing to the cells?

üì• Dataset - Public RNA-seq from GEO

Use a real published dataset:

  • Example GEO Series: GSE183947 (breast cancer drug treatment)
  • Data Type: Pre-computed count matrix
  • Format: Tab-delimited text file

1. Quality Control & Exploratory Analysis (20 points)

library(DESeq2)
library(ggplot2)

# Load data
counts <- read.table("counts.txt", header=TRUE, row.names=1)
metadata <- read.table("metadata.txt", header=TRUE)

# Create DESeq2 object
dds <- DESeqDataSetFromMatrix(
    countData = counts,
    colData = metadata,
    design = ~ condition
)

# Filter low counts
keep <- rowSums(counts(dds)) >= 10
dds <- dds[keep,]

# Normalize
dds <- estimateSizeFactors(dds)

# PCA plot
vsd <- vst(dds)
plotPCA(vsd, intgroup="condition")

# Sample correlation heatmap
library(pheatmap)
sampleDists <- dist(t(assay(vsd)))
pheatmap(as.matrix(sampleDists))

2. Differential Expression Analysis (30 points)

# Run DESeq2
dds <- DESeq(dds)
res <- results(dds, alpha=0.05)

# Extract significant genes
sig_genes <- subset(res, padj < 0.05 & abs(log2FoldChange) > 1)

# Separate up and down
up_genes <- subset(sig_genes, log2FoldChange > 1)
down_genes <- subset(sig_genes, log2FoldChange < -1)

# Summary statistics
summary(res)
table(sig_genes$log2FoldChange > 0)

Required Visualizations:

  • Volcano Plot: Show all genes, highlight significant
  • MA Plot: Log2FC vs. mean expression
  • Heatmap: Top 50 DE genes
  • PCA: Sample clustering

3. Pathway Enrichment Analysis (30 points)

library(clusterProfiler)
library(org.Hs.eg.db)

# Convert to Entrez IDs
up_entrez <- bitr(rownames(up_genes), 
                  fromType="SYMBOL", toType="ENTREZID",
                  OrgDb=org.Hs.eg.db)

# GO Enrichment
go_up <- enrichGO(gene = up_entrez$ENTREZID,
                  OrgDb = org.Hs.eg.db,
                  ont = "BP",
                  pvalueCutoff = 0.05)

# KEGG Pathway
kegg_up <- enrichKEGG(gene = up_entrez$ENTREZID,
                      organism = 'hsa',
                      pvalueCutoff = 0.05)

# Dot plot
dotplot(go_up, showCategory=20)
dotplot(kegg_up)

Required Analyses:

  • GO Enrichment (up-regulated genes)
  • GO Enrichment (down-regulated genes)
  • KEGG Pathway enrichment
  • Reactome pathway analysis (optional)

4. The Biological Story (20 points)

This is the key deliverable! Write a 1-2 page narrative explaining:

A. What's Being Activated?

  • What pathways are up-regulated?
  • What biological processes are enhanced?
  • What does this tell us about the drug's mechanism?

B. What's Being Suppressed?

  • What pathways are down-regulated?
  • Is this expected? (e.g., cell cycle arrest in cancer treatment)

C. Clinical Implications

  • Is the drug hitting its intended target?
  • Any unexpected effects?
  • Potential resistance mechanisms?
  • Biomarkers for patient selection?

D. Visual Summary

Create an infographic or pathway diagram showing the key story
