üéØ Challenge Overview

Analyze RNA-seq data from a cancer drug treatment experiment. Find differentially expressed genes, identify affected pathways, and tell the biological story of what's happening at the molecular level.

The Experiment

Cell Line: Human cancer cells (breast, lung, or other)

Treatment: Therapeutic compound vs. vehicle control

Time Point: Post-treatment (typically 24-48 hours)

Replicates: At least 3 control, 3 treated

Your Job: What is this drug doing to the cells?

üì• Finding Your Dataset

üéØ Your First Task: Dataset Selection

You'll search public repositories for RNA-seq data from a cancer drug treatment study. This tests your ability to find, evaluate, and work with real transcriptomic data.

Dataset Requirements

Find a dataset that meets these criteria:

  • Data Type: RNA-seq gene expression data
  • Format: Count matrix or normalized expression values
  • Experiment Design: Treatment vs. Control comparison
  • Replicates: At least 3 biological replicates per condition
  • Organism: Human (Homo sapiens) preferred
  • Context: Cancer cell lines with drug treatment

üîç Where to Search for RNA-seq Data

Option 1: GEO (Gene Expression Omnibus)

Primary repository for gene expression data: https://www.ncbi.nlm.nih.gov/geo/

Search strategies:

  • "RNA-seq" AND "cancer" AND "drug treatment"
  • "breast cancer" AND "chemotherapy" AND "RNA-seq"
  • "drug response" AND "transcriptome"
  • Filter by: Organism = Homo sapiens, Study type = Expression profiling by high throughput sequencing

Good dataset examples to search for:

  • GSE183947 - Breast cancer drug treatment
  • GSE164174 - Drug resistance in cancer
  • GSE149215 - Targeted therapy response

Option 2: TCGA (The Cancer Genome Atlas)

Large-scale cancer genomics data: https://portal.gdc.cancer.gov/

# Download using TCGAbiolinks
library(TCGAbiolinks)
library(SummarizedExperiment)

# Query for RNA-seq data

    project = "TCGA-BRCA",  # Or TCGA-LUAD, TCGA-COAD, etc.

    data.type = "Gene Expression Quantification",

# Download

# Extract count matrix

Note: TCGA data is organized by tumor vs. normal, not drug treatment. You'll need to select samples based on clinical data.

‚ö° What Makes a Good Challenge Dataset?

Look for datasets with these features:

  • Clear experimental design: Well-defined treatment and control groups
  • Good sample size: At least 3 replicates per condition (more is better)
  • Published studies: Check if there's an associated publication for context
  • Available count data: Raw or normalized counts, not just processed results
  • Interesting biology: Novel drugs, targeted therapies, or known mechanisms

Avoid:

  • Single-cell RNA-seq (different analysis methods)
  • Time-series with many time points (too complex)
  • Datasets with only processed data (no count matrix)
  • Experiments with confounded designs

üìù Document Your Dataset

In your final report, include:

  • GEO/SRA accession: GSE###### or SRP######
  • Study title and authors: What was the original research?
  • Experimental design: What treatment? What cancer type?
  • Sample information: How many replicates? What conditions?
  • Data format: Count matrix? Normalized? What units?
  • Download method: Commands or tools used

üìã Required Deliverables

1. Quality Control & Exploratory Analysis (20 points)

library(DESeq2)
library(ggplot2)

# Load data
counts <- read.table("counts.txt", header=TRUE, row.names=1)
metadata <- read.table("metadata.txt", header=TRUE)

# Create DESeq2 object
dds <- DESeqDataSetFromMatrix(
    countData = counts,
    colData = metadata,
    design = ~ condition
)

# Filter low counts
keep <- rowSums(counts(dds)) >= 10
dds <- dds[keep,]

# Normalize
dds <- estimateSizeFactors(dds)

# PCA plot
vsd <- vst(dds)
plotPCA(vsd, intgroup="condition")

# Sample correlation heatmap
library(pheatmap)
sampleDists <- dist(t(assay(vsd)))
pheatmap(as.matrix(sampleDists))

Required QC Checks:

  • Library size distribution across samples
  • Sample clustering (PCA or MDS)
  • Sample-to-sample correlations
  • Detection of outliers or batch effects

2. Differential Expression Analysis (30 points)

# Run DESeq2
dds <- DESeq(dds)
res <- results(dds, alpha=0.05)

# Extract significant genes
sig_genes <- subset(res, padj < 0.05 & abs(log2FoldChange) > 1)

# Separate up and down
up_genes <- subset(sig_genes, log2FoldChange > 1)
down_genes <- subset(sig_genes, log2FoldChange < -1)

# Summary statistics
summary(res)
table(sig_genes$log2FoldChange > 0)

# Top genes
top_genes <- head(sig_genes[order(sig_genes$padj),], 20)

Required Visualizations:

  • Volcano Plot: Show all genes, highlight significant
  • MA Plot: Log2FC vs. mean expression
  • Heatmap: Top 50 differentially expressed genes
  • Box plots: Expression of key genes across conditions

3. Pathway Enrichment Analysis (30 points)

library(clusterProfiler)
library(org.Hs.eg.db)

# Convert to Entrez IDs
up_entrez <- bitr(rownames(up_genes), 
                  fromType="SYMBOL", toType="ENTREZID",
                  OrgDb=org.Hs.eg.db)

# GO Enrichment
go_up <- enrichGO(gene = up_entrez$ENTREZID,
                  OrgDb = org.Hs.eg.db,
                  ont = "BP",
                  pvalueCutoff = 0.05)

go_down <- enrichGO(gene = down_entrez$ENTREZID,
                    OrgDb = org.Hs.eg.db,
                    ont = "BP",
                    pvalueCutoff = 0.05)

# KEGG Pathway
kegg_up <- enrichKEGG(gene = up_entrez$ENTREZID,
                      organism = 'hsa',
                      pvalueCutoff = 0.05)

# Visualize
dotplot(go_up, showCategory=20)
dotplot(kegg_up, showCategory=15)

# Optional: Reactome
library(ReactomePA)
reactome <- enrichPathway(gene = up_entrez$ENTREZID,
                         pvalueCutoff = 0.05)

Required Analyses:

  • GO Enrichment (Biological Process) for up-regulated genes
  • GO Enrichment (Biological Process) for down-regulated genes
  • KEGG Pathway enrichment
  • Create dot plots or bar plots for top enriched terms

4. The Biological Story (20 points)

This is the key deliverable! Write a 1-2 page narrative explaining:

A. What's Being Activated?

  • What pathways are up-regulated?
  • What biological processes are enhanced?
  • What does this tell us about the drug's mechanism?

B. What's Being Suppressed?

  • What pathways are down-regulated?
  • Is this expected? (e.g., cell cycle arrest in cancer treatment)
  • Any tumor suppressor genes affected?

C. Clinical Implications

  • Is the drug hitting its intended target?
  • Any unexpected effects or off-target activity?
  • Potential resistance mechanisms?
  • Biomarkers for patient selection?

D. Visual Summary

Create an infographic or pathway diagram showing the key story

üí° Analysis Strategy Tips

  1. Start with QC: Ensure samples cluster by condition, not batch
  2. Set reasonable thresholds: padj < 0.05 and |log2FC| > 1 is standard
  3. Consider effect size: High log2FC variants are more biologically meaningful
  4. Look for coherent pathways: Multiple genes in same pathway = stronger signal
  5. Read the original paper: Compare your findings to published results
  6. Think mechanistically: Connect pathways to known drug mechanisms
